How We Made "TKrecipies" Website :


Back to Home
  1. Made a directory folder in the terminal with “mkdir” TKrecipes
  2. Inside the folder we created various html files using vscode
  3. Typed in a “html 5” in vscode. This is a basic template
  4. Created a repository on GitHub 'TKrecipies'
  5. Initialised our local repository with the ‘git init’ command on the terminal.
  6. Added our website files to the repository with ‘git add’
  7. Used ‘git commit -m “initial commit”’ to commit changes
  8. We linked our local repository to the remote repository by copying the url from the GitHub page. Then used the commas ‘git remote add origin url’
  9. Pushed changes from local repository to Github with the command ‘git push origin main’
  10. Add Tomi as a collaborator to test out the cloning and branching features.
  11. Edited some recipies and made the page you're looking at now!
  12. Used W3C validator to validate our HTML codes. All good!
  13. Pushed all our changes into the main repository with the3 following commands: git add, git commit -m, git push.

Voila.
